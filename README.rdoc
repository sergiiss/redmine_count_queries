= Redmine Queries Issues Count

Redmine plugin to display issues count for queries in the sidebar.

== Installation
1. Follow the Redmine plugin installation steps at: http://www.redmine.org/wiki/redmine/Plugins and make sure the plugin is installed to +vendor/plugins/redmine_queries_issues_count+
2. Restart your Redmine web servers (e.g. mongrel, thin, mod_rails)
